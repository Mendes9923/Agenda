Conecte seu banco de dados Supabase

📋 Passo 1 — Crie sua conta
Acesse supabase.com e crie um projeto gratuito.
📋 Passo 2 — Crie as tabelas
No painel do Supabase, vá em SQL Editor e execute o script SQL disponível abaixo clicando em "📄 Ver Script SQL".
📋 Passo 3 — Copie as credenciais
Vá em Project Settings → API e copie:
• Project URL
• anon public key

As credenciais são salvas apenas no seu navegador (localStorage)
